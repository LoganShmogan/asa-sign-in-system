# SIGN-IN/OUT SYSTEM

## Project Outline:
(figma design https://www.figma.com/design/ibjI6hXJar9CuiHN5D4MLM/Industry-Project-Prototype?node-id=0-1&t=VmJs90Ldk9BvlUv5-1)
## Aim:

Deliver a sign- in/out system with a notice board as the first point of contact for the After School Adventures childcare.

## Outline:

The system will have an area for parents/caregivers/family (will just be called caregiver/s from now on) sign- in/out their child when dropping them off and picking them up. This will also include staff signing a child in/out when the are dropped and picked up from school. Along side this there will be a notice board that includes notifying the caregivers of interactions that need their attention or basic operations/activities at After School Adventures.

## Deliverables from project:

1. Computer setup
2. Notice Board
3. Notice Board Facebook integration
4. Admin/Manager/Staff roles
5. Staff sign-in/out + CRUD (Create / Read / Update / Delete)
6. Child sign-in/out + CRUD + Specific notifications
7. Holiday Programme booking
8. Xero integration
9. Web based Dashboard
10. Testing - Real world and controlled environment
